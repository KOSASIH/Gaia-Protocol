# Gaia Protocol

## Overview
A planetary DAO for quantum-managed resources. See simulations/ for AI/quantum code, contracts/ for blockchain logic.

## Setup
1. `npm install`
2. `pip install -r requirements.txt`
3. Run simulations: `python simulations/quantum_ledger.py`
4. Deploy: `npm run deploy`

## Vision
Perfect Allocation via AI and quantum sync.
